# SolidRun's Renesas Product-Line Build Scripts

This project is organised in branches for different BSP versions:

- [develop-vlp4](https://github.com/SolidRun/build_rzg2lc/tree/develop-vlp4):

  Based on Renesas Verified Linux Package v4, featuring:
  
  - U-Boot v2021.10 (for RZ/G2L & RZ/G2LC)
  - Linux v6.1
  
  This version is under active development.

- [develop-vlp3](https://github.com/SolidRun/build_rzg2lc/tree/develop-vlp3):

  Based on Renesas Verified Linux Package v3, featuring:
  
  - U-Boot v2021.10
  - Linux v5.10
  
  This version is considered stable and maintained with bug fixes only.
